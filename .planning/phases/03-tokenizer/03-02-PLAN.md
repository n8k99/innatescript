  <action>
Replace the `tokenize` stub in `src/parser/tokenizer.lisp` with a full position-based character loop.

**Implementation structure:**

1. Define internal helper functions using `labels` inside `tokenize`:
   - `(current)` — return char at pos, or nil if at end
   - `(advance)` — consume one char, update pos/line/col. On `\n`: incf line, setf col 1. On other: incf col. Return the consumed char.
   - `(peek-next)` — return char at pos+1, or nil if out of bounds
   - `(emit (type &optional value))` — push `(make-token :type type :value value :line start-line :col start-col)` onto tokens list. NOTE: emit must use pre-captured start-line/start-col, not current line/col.
   - `(skip-whitespace)` — consume spaces and tabs (NOT newlines) silently

2. Main loop: `(loop while (current) do ...)` with a `case` dispatch on `(current)`:

   Single-char tokens (capture start-line/start-col before advance, then emit):
   - `#\]` -> `:rbracket`
   - `#\(` -> `:lparen`
   - `#\)` -> `:rparen`
   - `#\{` -> `:lbrace`
   - `#\}` -> `:rbrace`
   - `#\:` -> `:colon`
   - `#\,` -> `:comma`
   - `#\#` -> `:hash`
   - `#\/` -> `:slash`
   - `#\+` -> `:plus`
   - `#\@` -> `:at`

   Two-char operators (peek-next before consuming):
   - `#\!` -> if peek-next is `#\[`, advance twice, emit `:bang-bracket`. Else signal `innate-parse-error`.
   - `#\|` -> if peek-next is `#\|`, advance twice, emit `:pipe-pipe`. Else signal `innate-parse-error`.
   - `#\-` -> if peek-next is `#\>`, advance twice, emit `:arrow`. Else signal `innate-parse-error`.

   `#\[` handling (STUB for now — Plan 03 adds wikilink disambiguation):
   - For this plan, just emit `:lbracket` and advance. Plan 03 will add the `[[` lookahead.

   Whitespace:
   - `#\Space`, `#\Tab` -> advance (skip silently)
   - `#\Newline` -> advance (STUB: just skip for now; Plan 03 adds :newline emission with collapse logic)

   `#\"` -> call `%read-string` helper (see Task 2)
   `#\<` -> call `%try-emoji-slot` helper (see Task 2)
   digit-char-p -> call `%read-number` helper (see Task 2)
   alpha-char-p or `#\_` -> call `%read-bare-word` helper (see Task 2)

   Otherwise -> signal `innate-parse-error` with current line/col and the unexpected character.

3. Return `(nreverse tokens)`.

**Key position tracking rule:** At the top of each dispatch branch, capture `(let ((sl line) (sc col)) ...)` BEFORE calling advance. Use `sl`/`sc` when pushing the token.

Add helper functions inside the `labels` block of `tokenize` in `src/parser/tokenizer.lisp`:

1. **`%read-string`**: Called when current char is `#\"`.
   - Capture start-line/start-col at the opening `"`.
   - Advance past opening `"`.
   - Accumulate chars into a list (`buf`):
     - If current is nil (EOF): signal `(error 'innate-parse-error :line start-line :col start-col :text "Unterminated string literal")`
     - If current is `#\\`: advance (consume backslash), then check current: if `#\"` push `#\"` and advance; if `#\\` push `#\\` and advance; else signal `innate-parse-error` with "Unknown escape".
     - If current is `#\"`: advance (consume closing quote), push token `(make-token :type :string :value (coerce (nreverse buf) 'string) :line start-line :col start-col)`, return.
     - Otherwise: push current to buf, advance.

2. **`%read-number`**: Called when current char satisfies `digit-char-p`.
   - Capture start-line/start-col.
   - Accumulate consecutive digit chars into a list.
   - Emit `(make-token :type :number :value (coerce (nreverse buf) 'string) :line start-line :col start-col)`.

3. **`%read-bare-word`**: Called when current char satisfies `alpha-char-p` or is `#\_`.
   - Capture start-line/start-col.
   - Accumulate chars matching `[a-zA-Z0-9_]` (alphanumericp or `#\_`).
   - Convert to string.
   - If string equals `"decree"`, emit with type `:decree`.
   - Otherwise emit with type `:bare-word`.
   - Value is always the accumulated string.

4. **`%try-emoji-slot`**: Called when current char is `#\<`.
   - Check if `source[pos..pos+6]` equals `"<emoji>"` using `(and (>= (- len pos) 7) (string= "<emoji>" source :start2 pos :end2 (+ pos 7)))`.
   - If match: capture start-line/start-col, advance 7 times, emit `(make-token :type :emoji-slot :value "<emoji>" :line sl :col sc)`.
   - If no match: signal `innate-parse-error` with current line/col and "Unexpected character <".

Wire these helpers into the main `case` dispatch:
- `#\"` -> call `%read-string`
- `digit-char-p` check -> call `%read-number` (use `t` branch of case with a `cond` or a separate `cond` after the `case`)
- `alpha-char-p` or `#\_` check -> call `%read-bare-word`
- `#\<` -> call `%try-emoji-slot`
