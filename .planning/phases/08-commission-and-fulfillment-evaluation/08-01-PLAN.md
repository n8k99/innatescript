  <action>
Replace three stub cases in the `eval-node` etypecase in `src/eval/evaluator.lisp`:

**:emission case (replacing lines 133-135):**
```lisp
((eql :emission)
 (let ((children (node-children node)))
   (if (= (length children) 1)
       (eval-node (first children) env)
       (mapcar (lambda (child) (eval-node child env)) children))))
```
Single child returns the value directly. Multiple children returns a list of evaluated values. This matches CONTEXT.md: "Return the list of evaluated values (or the single value if only one child)."

**:wikilink case (replacing lines 137-140):**
```lisp
((eql :wikilink)
 (let* ((title (node-value node))
        (result (resolve-wikilink (eval-env-resolver env) title)))
   (if (resistance-p result)
       (signal 'innate-resistance
               :message (resistance-message result)
               :source (resistance-source result))
       (innate-result-value result))))
```
Same pattern as :reference resolver fallthrough — call generic, check resistance-p, signal or unwrap.

**:bundle case (replacing lines 121-124):**
When standalone (not adjacent to :agent — that adjacency is handled in Plan 02):
```lisp
((eql :bundle)
 (let* ((name (node-value node))
        (nodes (load-bundle (eval-env-resolver env) name)))
   (if nodes
       ;; Evaluate returned AST nodes as sub-program, return last result
       (let ((last-result nil))
         (dolist (n nodes)
           (setf last-result (eval-node n env)))
         last-result)
       ;; Bundle not found — signal resistance
       (signal 'innate-resistance
               :message (format nil "Bundle not found: ~a" name)
               :source (or name "bundle")))))
```
load-bundle returns a list of AST nodes or nil. If nodes, evaluate each and return the last result (progn semantics). If nil, signal resistance.

NOTE: The :bundle node can also appear with `(node-value node)` being nil when it comes from complex paren groups (the parser makes a :bundle with :children but nil :value). In that standalone context (no agent adjacency), if value is nil, just evaluate children as expressions and return last result. But this edge case is handled in Plan 02 when commission adjacency is implemented.
