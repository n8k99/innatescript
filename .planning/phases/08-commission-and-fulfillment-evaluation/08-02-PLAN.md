**3. Replace :fulfillment stub in eval-node (line 129-132):**
```lisp
((eql :fulfillment)
 (let ((left (first (node-children node)))
       (right (second (node-children node))))
   (handler-case
       (eval-node left env)
     (innate-resistance (c)
       (declare (ignore c))
       (eval-node right env)))))
```
This is the core of Innate's error model. handler-case wraps ONLY the left-side evaluation. If left signals innate-resistance, catch it and evaluate the right side instead. If left succeeds, return its value — right is never evaluated.

**4. Modify `evaluate` pass-2 loop for commission adjacency detection:**
Replace the `dolist` loop with index-based iteration using `loop for i from 0 below (length children)`. When a non-decree child is an :agent node AND the next child (i+1) is a :bundle node, consume both and call deliver-commission. Otherwise evaluate normally.

```lisp
;; Pass 2: evaluate with commission adjacency detection
(let ((results nil)
      (len (length children))
      (children-vec (coerce children 'vector)))
  (loop for i from 0 below len
        for child = (aref children-vec i)
        do (unless (eq (node-kind child) :decree)
             (cond
               ;; Commission adjacency: :agent followed by :bundle
               ((and (eq (node-kind child) :agent)
                     (< (1+ i) len)
                     (eq (node-kind (aref children-vec (1+ i))) :bundle))
                (let* ((agent-name (node-value child))
                       (bundle-node (aref children-vec (1+ i)))
                       (instruction (or (node-value bundle-node)
                                        ;; Complex bundle: evaluate children to build instruction string
                                        (format nil "~{~a~^ ~}"
                                                (mapcar (lambda (c) (eval-node c env))
                                                        (node-children bundle-node)))))
                       (result (deliver-commission (eval-env-resolver env)
                                                    agent-name instruction)))
                  (push (if (resistance-p result)
                            (progn
                              (signal 'innate-resistance
                                      :message (resistance-message result)
                                      :source (resistance-source result))
                              nil)
                            (innate-result-value result))
                        results)
                  (incf i))) ;; Skip the bundle node (loop will also incf)
               ;; Normal evaluation
               (t
                (let ((result (eval-node child env)))
                  (push result results))))))
  (nreverse results))
```
Key: The `(incf i)` inside the commission branch plus the loop's natural increment skips the :bundle node that was consumed as part of the commission.
